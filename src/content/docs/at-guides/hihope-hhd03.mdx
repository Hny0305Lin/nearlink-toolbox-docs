# 润和 HH-D03 AT指令指南

<Card title="设备信息" icon="information">
  - 设备型号：HiHope HH-D03
  - 主控芯片：Hi3861
  - 通信方式：UART
  - 波特率：115200
  - 数据位：8
  - 停止位：1
  - 校验位：无
</Card>

## 基础 AT 指令

<Tabs>
  <TabItem label="查询版本">
    ```bash
    AT+VER
    ```
    返回固件版本信息
  </TabItem>

  <TabItem label="恢复出厂">
    ```bash
    AT+RESTORE
    ```
    恢复出厂设置
  </TabItem>
</Tabs>

## 网络配置指令

<Tabs>
  <TabItem label="扫描网络">
    ```bash
    AT+SCAN
    ```
    扫描可用网络
  </TabItem>

  <TabItem label="连接网络">
    ```bash
    AT+CONN
    ```
    连接到指定网络
  </TabItem>
</Tabs>
