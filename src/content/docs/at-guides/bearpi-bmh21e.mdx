# 小熊派 BM H21E AT指令指南

<Card title="设备信息" icon="information">
  - 设备型号：BearPi BM H21E
  - 主控芯片：Hi2121
  - 通信方式：UART
  - 波特率：115200
  - 数据位：8
  - 停止位：1
  - 校验位：无
</Card>

## 基础 AT 指令

<Tabs>
  <TabItem label="查询版本">
    ```bash
    AT+VER
    ```
    返回固件版本信息
  </TabItem>

  <TabItem label="恢复出厂">
    ```bash
    AT+FACTORY
    ```
    恢复出厂设置
  </TabItem>
</Tabs>

## 网络相关指令

<Tabs>
  <TabItem label="配置网络">
    ```bash
    AT+NETCFG
    ```
    配置网络参数
  </TabItem>

  <TabItem label="查询状态">
    ```bash
    AT+STATUS
    ```
    查询网络状态
  </TabItem>
</Tabs>
