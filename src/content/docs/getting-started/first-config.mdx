## 烧写设置

烧写设置, 有两个配置选项, 分别为: 烧写工具配置、波特率设置。

### 烧写工具配置

烧写模式分为两种, 分别为: 海思模式、自定义模式。

- 海思模式: 使用海思Burntool工具进行烧写, 需要您下载[Burntool工具](https://cucc.tj.fz.fj.nearlink-firmwares-mirrors.haohanyh.ovh:44443/?tools/burntool)后, 在**烧写工具**里配置好可执行文件存储路径。

- 自定义模式: 使用开源ws63flash工具进行烧写, 需要您下载到[ws63flash工具](https://github.com/goodspeed34/ws63flash/releases)后, 在**烧写工具**里配置好可执行文件存储路径。

**最新版1.7.0版本支持自动上电烧写, 依赖海思Burntool工具能力, 这里推荐需要自动上电用户, 选择海思烧写工具**

### 波特率设置

这里的波特率配置会影响到烧写全局使用([浏览固件页的一键烧写](/getting-started/started#一键烧录)、[固件烧写](/getting-started/started#手动烧录)、[批量烧写](/getting-started/started#批量烧录)), 请您根据您实际使用情况进行配置。

<Aside type="danger">Hi2821 芯片的固件烧写为确保稳定性需使用 115200 波特率, 其余情况自行选择使用。</Aside>

## 串口设置

配置串口通信的默认参数，包括波特率、数据位等(在您不了解配置的情况下, 默认配置就是最好的)

默认配置如下:

<Card title="串口配置" icon="information">
  - 波特率: 115200(推荐)
  - 数据位: 8
  - 校验位: 无
  - 合包延迟: 50(推荐)
</Card>

默认选项如下: 

<Card title="默认选项" icon="information">
  - 自动滚动: 开启（当接收到新数据时，自动滚动到最新数据）
  - 自动换行: 开启（发送数据时自动在末尾添加换行符, 确保串口工具能正常解析）
  - 显示时间戳: 开启（在每条接收的数据前显示时间）
  - 显示延迟: 关闭（显示板载与用户之间的消息时间间隔）
  - 快捷命令自动发送: 开启（点击后跟自动换行一个效果, 自动发送）
</Card>
