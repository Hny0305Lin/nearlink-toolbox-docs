星闪工具箱的初次开始，需要您做好如下准备工作:

- 准备一台电脑, **64位 Windows 10 或更高版本**
- 准备一对星闪开发板, 如[BearPi-Pico H3863](https://bearpi.cn/core_board/bearpi/pico/h3863/)
- 准备好CH340沁恒驱动安装, 在下面教程我们会提供官网链接
- 准备两根Type-C数据线, 随时为星闪开发板接入计算机做准备

<Aside type="tip" title="开发者提示">如果您想成为一名开发者, 我们更推荐您安装工具箱的同时, 配套安装开发环境!</Aside>

- [安装Hispark Studio](/getting-started/started#安装hispark-studio)
- [安装Git](/getting-started/started#安装git)
- [编译](/getting-started/started#sdk代码下载--编译) & 烧录 & 测试
- 配置加速编译(海星通社区待施工)
- 安装VSCode Deveco Device Tool插件(可选,待施工)
- 下载保留DevTools_CFBB编译工具链(可选,待施工)
- 下载浩瀚银河星闪工具箱插件(待施工)
- 下载浩瀚银河星闪工具箱开发资料(待施工)

## 开始下载并安装

下载并安装以下软件:

<CardGrid>
  <Card title="CH340驱动安装" icon="list-format">
    <LinkCard title="点我下载" href="//www.wch.cn/downloads/CH341SER_EXE.html" target="_blank"/>
  </Card>
  <Card title="星闪工具箱安装" icon="list-format">
    <LinkCard title="点我下载" href="//nearlink.sgguo.com/" target="_blank"/>
  </Card>
</CardGrid>

下载在您可以知晓目录位置，比如"downloads", "下载文件夹"这些存储在非 Windows系统盘 的文件夹。

### CH340驱动安装

从沁恒官网下载到`CH341SER.EXE`文件, 保存后打开, 如果遇到UAC弹窗请点击"是"放行操作。

![用户账号控制窗口，请放行](https://s2.loli.net/2025/01/03/EDmvPrc5i7etsjQ.jpg)

然后我们就能打开页面了。

<Aside type="danger">我们推荐新手用户, 安全使用电脑, UAC弹窗是保护您计算机安全的行为, 请不要因此在网络上寻找关闭弹窗的办法。</Aside>

遇到如下页面, 请点击"安装"按钮, 程序会自动运行, 直至弹窗"程序预安装成功"您就完成了CH340驱动安装。

![CH340驱动安装页面](https://s2.loli.net/2025/01/03/NIvo5zYUG1OFu2E.jpg)

![CH340驱动安装提示](https://s2.loli.net/2025/01/03/ulKfa36qe9gwntz.jpg)

如果遇到安装失败了, 请在打开`CH341SER.EXE`文件之前, 先关闭所有杀毒软件, 鼠标右击选中"以管理员身份运行(A)"。

### 星闪工具箱安装

星闪工具箱的安装, 请前往官网[星闪工具箱](https://nearlink.sgguo.com/)下载安装包。

![星闪工具箱官网](https://s2.loli.net/2025/01/03/oVfW5JmpQGxubva.jpg)

<Aside type="danger">下载好运行exe可执行文件, 这一步可能也会遇到UAC弹窗, 请点击"是"放行操作。</Aside>

请选择中文(简体), 点击OK开始安装。

![星闪工具箱安装1](https://s2.loli.net/2025/01/03/XHLG2rw8zsAOjKJ.jpg)

点击下一步继续安装。

![星闪工具箱安装2](https://s2.loli.net/2025/01/03/BjXIkLd32VHmqNO.jpg)

安装目录请修改成自己的非 Windows系统盘 的文件夹，点击下一步继续安装。

![星闪工具箱安装3](https://s2.loli.net/2025/01/03/gaNk5riQ3OJtpHe.jpg)

点击下一步完成最后安装。

![星闪工具箱安装4](https://s2.loli.net/2025/01/03/XrMOHhzp7tkf3Fo.jpg)

勾选"运行 NearLink Toolbox"和"创建桌面快捷方式", 点击完成。

![星闪工具箱安装5](https://s2.loli.net/2025/01/03/tI3db5V4QJKCGNF.jpg)

软件会自动开启, 如果遇到这样页面正常启动, 恭喜您完成了安装使用。

![星闪工具箱启动](https://s2.loli.net/2025/01/03/rCB9IbFGXKDWacd.jpg)

### 安装Git

<Aside type="tip" title="浩瀚银河提示">浩瀚银河为每一位开发者服务, 在教程内, 我们**更推荐您使用Git, 并在开发期间学会使用Git来管理您的项目**, 具体请自行搜索学习, 这里提供的是Windows 下的 Git的安装!</Aside>

可以到 Git 的页面上下载 exe 安装文件并运行：

<LinkCard title="点我前往下载" href="//git-scm.com/download/win" target="_blank"/>

下载后，双击安装包，打开界面如下所示，点击 "next" 按钮开始安装：

完成安装之后，就可以使用命令行的 git 工具（已经自带了 ssh 客户端）了，另外还有一个图形界面的 Git 项目管理工具。

在开始菜单里找到 "Git"->"Git Bash"，会弹出 Git 命令窗口，你可以在该窗口进行 Git 操作。

### SDK代码下载 & 编译

通过Git指令下载, 在您的存储盘, 鼠标右键选择“Open Git Bash here”, 如图所示。

![image-20240807142741095.png](https://s2.loli.net/2025/01/04/YsSD8gJewRuc6oT.png)

在命令框中输入`git clone https://gitee.com/HiSpark/fbb_ws63.git`指令下载代码，等待下载完成。

![image-20240807151920249.png](https://s2.loli.net/2025/01/04/aRdzqBQIjOT1DnN.png)

<Aside type="note" title="注意">
    如果您遇到了相关的报错问题, Git大概率的问题频发于网络通信, 请关掉代理再进行尝试。
    
    或者就是请搜索软件自助扩大Git缓冲区 or 请携带报错内容善于使用搜索引擎自助解决!
</Aside>

下载完成后, 就是在文件夹如图显示。

![image-20240801103855026.png](https://s2.loli.net/2025/01/04/qBD45Ac2e6EhjfC.png)

代码下载完成后, 打开HiSpark studio工具, 点击“新建工程”。

![image-20240801104223178.png](https://s2.loli.net/2025/01/04/oa2UseR1GjxkMQB.png)

新建工程界面中芯片: `ws63`, 工程名: "xxx"(用户自定义, 但是不能带中文, 特殊符号), 软件包: `xxx/fbb_ws63/src`(SDK软件包存放路径, 这个地方一定要选到src层级, 否则新建会失败), 配置选择完成后，点击“完成按钮”。

![image-20240801112357075.png](https://s2.loli.net/2025/01/04/ubZ2wShPO7m6lzc.png)

点击“build”或者“rebuild”编译, 等待编译完成。

![image-20240801112427220.png](https://s2.loli.net/2025/01/04/AXK7NElon3vUbM1.png)
