## NearLink ToolBox  

<CardGrid stagger>  
	<Card title="Team Embedded Masterpieces" icon="rocket">  
		This project aims to bridge the gap between NearLink users and developers, allowing users to easily use open-source firmware while enabling developers to release updates promptly.  
	</Card>  
	<Card title="Supported Development Board Devices" icon="rocket">  
		Supports HiHope, BearPi, Tiangonix, Haohanyh manufacturers, and is continuously working on supporting more development boards and products from the open-source community.  
	</Card>  
	<Card title="Independent of Frontend" icon="rocket">  
		Tauri supports any frontend, allowing the team to develop with ease while ensuring seamless interconnectivity between software functions.  
	</Card>  
	<Card title="Secure Communication" icon="rocket">  
		Uses TanStack Router for communication, ensuring secure front-end and back-end communication under any circumstances within the software.  
	</Card>  
	<Card title="Low Performance Overhead" icon="rocket">  
		Thanks to the performance of Rust, the software's backend functionality achieves an extremely low 4MB memory usage on Windows.  
	</Card>  
	<Card title="Cross-platform Use" icon="rocket">  
		Supports Windows, MacOS, and Ubuntu platforms, allowing users to access it anytime, anywhere.  
	</Card>  
</CardGrid>
