## Preparation

<Tabs>
  <TabItem label="HiHope AT">
    <Steps>
    1. Connect the HiHope HH-D03 development board to the computer, open the NearLink ToolBox, and check if it connects properly.
    ![WeChat_20250104223851.png]()
    2. In the firmware page, select **HiHope HH-D03(BS21) AT firmware**, click to find the file page, then click **One-click Flash**, and confirm by clicking **Confirm Flash** after checking the download and flash information.
    ![WeChat_20250104224426.png]()
    3. After following the process, press the button on the board to flash the AT firmware. Once the flashing is complete, you are done.
    4. Continue in the file page, find the button on the right of **Hihope HH-D03 Multi-Command Set Professional Edition.json**, click **One-click Import**, then click **Confirm Download**.
    ![WeChat_20250104224950.png]()
    5. Once everything is ready, go to the **Serial Terminal** page, configure it, then click connect. Test the **AT** command, and if it returns OK, it’s a preliminary success, as shown:
    ![WeChat_20250105002543.png]()
    6. Do not leave the current page of the software, click to go to the [HiHope HH-D03 AT Usage](/at-guides/hihope-hhd03#润和-at-指令使用教程) page to check the detailed usage guide.
    </Steps>
